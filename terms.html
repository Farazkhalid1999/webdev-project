Terms


CSS -also known as Cascading style sheets are used to define the look and feel of a website

Cache -Cached files are those that are saved or copied (downloaded) by a web browser.

Back-End -The back end of a website is the part hidden from view of regular website visitors.

Bandwidth -Bandwidth can refer to two different things: the rate at which data can be transferred or the total amount of data allowed to be transferred.

Browser -Browser refers to the program a website visitor is using to view the web site.

DNS -Stands for Domain Name Service. It is the thing that converts IP addresses into domain names.

DOCTYPE -The doctype declaration specifies which version of HTML is used in a document.

Domain -The domain is the name by which a website is identified. The domain is associated with an IP address.

Element -an element is the central building block of any document. Individual elements can contain text, other elements, or both.

Font Family -Font family is a group designation for defining the typefaces used in CSS documents.

Font Style -In CSS, the font style refers solely to whether a font is italic or not.

Font Weight -The font weight refers to how thick or thin (bold or light) a font looks.

Font End -It is all the components of a website that a visitor to the site can see (pages, images, content, etc).

GUI -A GUI uses an input device (like the mouse) and visual representations of how the user is able to interact with a web application.

HTML -Stands for Hypertext Markup Language. It is the primary language used to write web pages.

HTTP -Stands for Hyper Text Transfer Protocol.

HyperLink -A hyperlink is a link from one web page to another.

Inline Style -Elements with CSS written directly around the element it affects, instead of in a separate style sheet or header style.

Landing Page -A landing page is the page where a visitor first enters a website.

Meta Data -Meta data is the data contained in the header that offers information about the web page that a visitor is currently on.

Plug-In -A plug/in is a bit of third party code that extends the capabilities of a website.

Property -Property is a CSS term and is roughly equivalent to an HTML tag.

Script -Generally refers to a portion of code on an HTML page.

Selector -In CSS, the selector is the item a style will be applied to.

Tag -A tag is a set of markup characters that are used around an element to indicate its start and end.

Template -A template is a file used to create a consistent design across a website.

URL -Stands for Uniform Resource Locator.

Web Page -A web page is a single document, generally written in HTML/XHTML.

XHTML -Stands for Extensible Hypertext Markup Language.

XML -Stands for Extensible Markup Language.

Alignment -The positioning of the elements in your design (e.g. text, images, etc.).

Compression -Reducing a file size by eliminating excess data.

Hex Code -A code used in HTML and CSS to designate a specific color.

JPEG -An image file type that uses lossless compression.

Pixel -The smallest element of an image on a computer.

PPI -(Pixel Per Inch) another measure of image resolution, according to how many pixels are present within a given section of the image.

PNG -An image file format that's best used when the image has large areas of uniform color, or a transparent background (unlike JPEG).

Resolution -A way of measuring the sharpness and level of detail in an image.

Typeface -A design collection of characters, including letters, numbers, and punctuations. Examples include Times New Roman, Helvetica, and Arial.

Weight -In typefaces, the thickness of the strokes width. Some examples include demibold, light, and bold.

White Space -The blank space surrounding an object in design also see negative space.

Negative Space -The empty space surrounding a design, whether a webpage or single image.

Animation -The process by which a set of images simulate movement when they are shown in a specific order.

Bit -Abbreviation for binary digit. A bit is a unit of data that can have a value of 0 or 1..

Cookies -data saved by your web browser that identifies you or your preferences.

FTP -a common method of transferring files from one computer to another over the Internet.

RGB -(stands for Red, Green, Blue) RGB is the color model of digital graphics.

Hexa-decimal -Also referred to a hex numbers, they are a base-16 numbering system used to define colours online.

Drop-Down -A drop-down can be one of two elements: A drop-down menu is a navigational menu that has sub-menus or categories below it.

Wireframe -simplistic drawings and/or layouts of a webpage, which typically have no design applied.
