Web Developer


As a Web Developer you are responsible for designing, coding and modifying websites from the layout to the function. The website has to be according to a clients specification. The entry requirements usually for a web developer is to have some knowledge of programming in different computer languages such as Javascript, CSS and HTML. Having an IT degree or any relevant degree to do with computing helps a lot and can land you in senior positions. Many skills are required to become a web developer such as a good eye for design and being creative, excellent problem solving skills and being able to learn quickly.

Games Developer


As a Games Developer, you are responsible for creating games for either the public or certain clients. Like a Web Developer a degree in IT is needed or something similar such as software engineering. Game Developers like Web Developers need to have knowledge in coding however Game Developers need to be very good at designing and need to be creative as the nature of their job requires the person to think outside of the box to come up with new ideas for their game. Game Developers do not just focus on coding and designing they also do many other tasks such as creating story lines for their games, creating prototypes or beta versions of their game for people to test on, creating role-play mechanics etc.

Software Developer


Unlike Web or Game Developers, Software Developers create different types of software for the business or specific clients and do not focus on creating one thing. Depending on which business the software developer works in they may have to create different types of software ranging from security software to everyday software used by the public. Usually like the other jobs having a degree in an IT related field helps to gain a high salary job. Main skills to become a Software Developer is to have good math skills and having a logical mind, being able to work under pressure, good problem solver etc.
